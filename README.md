# test_jmc
Tes Teknis Programmer

`cd test_jmc`

#### Tahap Install
1. `$ cp .env.example .env`
2. `$ composer install`
3. `$ npm install`
4. `$ php artisan key:generate`
5. `$ php artisan migrate:fresh --seed`

### jalankan pada terminal / cmd terpisah
6. `$ npm run dev`
7. `$ php artisan serve`
